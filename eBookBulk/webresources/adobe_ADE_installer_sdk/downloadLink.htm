Step 1 of 4: Install ADE

  • Click the Install and Read button to the right.
  • You will be taken through the steps necessary to install Adobe Digital Editions on your computer.
  • If prompted, save the 'qaccs' file to your computer.
  • If you having problems installing ADE, try installing it manually.

Step 2 of 4: You are about to install ADE

  • Choose the Install button.
  • Wait for sometime to get a confirmation prompt.
  • If you having problems installing ADE, try installing it manually.

Step 3 of 4: Installing ADE

  • Click the Yes button.
  • You are now going to install ADE.
  • ADE Setup box will appear.
  • Check/Uncheck the components you don't want to install.
  • After installtion, Internet Explorer will display a yellow information bar.
  • If you having problems installing ADE, try installing it manually.

Step 4 of 4: Download your eBook

  • After installtion, Internet Explorer information bar will appear at the top of the window.
  • Click the bar and select Download File from the menu options.
  • A Download Item button will then appear to the left.
  • Click on the button to download your ebook.
  • If you having problems installing ADE, try installing it manually.
